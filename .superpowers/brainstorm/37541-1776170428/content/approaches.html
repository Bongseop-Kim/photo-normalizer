C

문서 기반 앱 (NSDocument)

배치 세션을 문서로 저장·복원. 여러 작업 세션 관리 가능

장점

  • App Store 심사에 가장 유리
  • 재처리·이력 관리 자연스러움
  • iCloud 자동 백업 가능

단점

  • NSDocument 아키텍처 러닝커브 높음
  • 현재 CLI 구조와 가장 거리가 멀음
  • 지금 당장 필요한 기능은 아님
추천: A (마법사) → 나중에 B로 확장
